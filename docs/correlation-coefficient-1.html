Correlation Coefficient

As indicated earlier, the strength of the linear association between two variables is mathematically measured with the so-call Correlation Coefficient. At times, it is also called the Pearson product-moment correlation coefficient, after Karl Pearson, who is credited with formulating r.

Mathematically, the correlation coefficient, \(r\), is calculated with the following equation:

\[\begin{equation} r = \frac{1}{(n-1)} *\sum_{i=1}^n \frac{x -\bar{x}}{Sx}*\frac{y -\bar{y}}{Sy} \end{equation}\]

Let’s break the equation into its components to see what is doing. Lets start by considering the product of the differences between each x value to the mean of x (\(\bar{x}\)) and each value of y to the mean of y (\(\bar{y}\)). Basically, this part:

\[\begin{equation} (x -\bar{x}) * (y -\bar{y}) \end{equation}\]

If all the different pairs of XY values are similarly larger or smaller than their means, then the product will be positive. Like the image below.

Figure 5.32: Positive Correlation

If for each pair of XY values, the x-value is smaller than the y-value, then the product will be negative. Like the image below.

Figure 5.33: Negative Correlation

If your pairs of XY values, include an assortment of low x with high y, and vic, then you should expect little or no correlation, like the image below.

Figure 5.34: No Correlation

Next in the equation, we divide the products of the differences for each pair of values, by the standard deviations, \(S\), of \(x\) and \(y\). This is done, such that the results can be unitless. Finally, you take the average of all the products. Basically, on average, how x and y relate to each other.

The formula for the correlation coefficient is technically difficult to work with because of all the subtractions and products. However, if you were to work the math, it can be exactly defined with a much simpler equation to calculate, but more difficult to interpret:

\[\begin{equation} r = \frac{n * \sum_{} xy- (\sum_{} x)*(\sum_{} y)}{\sqrt {n * \sum_{} x^2- (\sum_{} x)^2 } * \sqrt {n * \sum_{} y^2- (\sum_{} y)^2 }} \end{equation}\]

Be careful! The notation \(\sum_{}x^2\) means first square \(x\) and then calculate the sum, whereas \((\sum_{}x)^2\) means first sum the x values and then square the result.

For the equation above, all we have to compute \(\sum_{}x\), \(\sum_{}y\), \(\sum_{}x^2\), \(\sum_{}y^2\), and \(\sum_{}x*y\), which is very simple using any scientific calculator and even simpler using R. Let try, for the sake of doing one full calculation of the coeficient of correlation and for you to use some tools from R.

Y=Data$TRUMPVoteAsFraction
X=Data$BachelorDegreePerStateAsFraction
SumX= sum(X) #sum all values of x
SumY= sum(Y)              #sum all values of y
SumX2=sum (X^2) #sum all values of x^2..
SumY2=sum (Y^2)
SumXY=sum (X*Y)  #sum all value of x * y
n=nrow(Data) # the number of observations is basically the number of rows in the database

The results are:

\(\sum_{}x\) = 5.48

\(\sum_{}y\) = 23.63

\(\sum_{}x^2\) = 0.69

\(\sum_{}y^2\) = 12.09

\(\sum_{}x*y\) = 2.45

n = 49

Now we plug those values into the coefficient of correlation, r, equation:

.

\[\begin{equation} r = \frac{n * \sum_{} xy- (\sum_{} x)*(\sum_{} y)}{\sqrt {n * \sum_{} x^2- (\sum_{} x)^2 } * \sqrt {n * \sum_{} y^2- (\sum_{} y)^2 }} \end{equation}\]

.

\[\begin{equation} r = \frac{49 * 2.45- (5.48)*(23.63)}{\sqrt {49 * 0.69- (5.48)^2 } * \sqrt {49 * 12.09- (23.63)^2 }} \end{equation}\]

.

In R, it is basically:

r=(n*SumXY-(SumX*SumY))   /  ( sqrt(n*SumX2 -(SumX^2))  * sqrt(n*SumY2 -(SumY^2))   )

\[\begin{equation} r = -0.83 \end{equation}\]

Alternatively, you can use the function “cor” in R, which calculates the coefficient correlation, directly. Like this:

r= cor(X,Y)

round (r, 2) #lets round the result to 2 digits
## [1] -0.83
